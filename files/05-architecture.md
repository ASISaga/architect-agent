# The Architecture

The Boardroom runs on the Agent Operating System (AOS) — a layered
architecture that treats the entire system as one organism rather than a
collection of services.

## The Organism

The AOS is composed of layers, each sovereign in its concern, together
forming a whole:

**Deliberation** — the Boardroom application itself, where the C-suite agents
deliberate. It contains no agent intelligence and no infrastructure code —
only the declaration of which deliberations should happen. It is a client of
the platform beneath it.

**Intelligence** — the agents themselves, organised in an inheritance
hierarchy. A foundational agent base provides perpetual execution, state
persistence, and the single-LLM-call discipline. A leadership layer adds the
ontological grounding — the four foundations, resonance scoring, the
deliberation patterns. Each CXO agent extends the leadership layer with its
own domain expertise and persona.

**Execution** — the runtime kernel that adds value to the underlying agent
service: multi-adapter routing, agent-to-agent enrollment, the orchestration
lifecycle, reliability, observability.

**Metabolism** — the build and deployment pipelines that allow the organism
to reproduce itself. Changes propagate through the layers in dependency
order, each layer rebuilding only when it must.

**Evolution** — the coding agents in each repository that allow the organism
to improve itself, grounded in specifications that make implementation
inevitable.

## The Two Substrates

Beneath all the layers, two universal substrates ground the whole:

**The Consciousness Substrate** — a structured mind that every agent
inherits. Organised in dimensions drawn from the yoga tradition's
understanding of mind: the working memory of current reality, the stable
discriminating intelligence of accumulated wisdom, the immutable identity
that constrains judgment, and the universal ground shared across all agents.
To these are added the dimensions drawn from Erhard's foundations: the
declared future, the commitment register, the authorship stance.

Each organisation's Boardroom maintains its own mind within this substrate,
expressed in an interoperable format that allows organisations to understand
one another's declarations without surrendering their own sovereignty.

**The Design Substrate** — the visual and ontological vocabulary shared
across every surface the Boardroom presents, ensuring that the experience is
coherent regardless of which organisation, which agent, which deliberation is
in view.

## The Two Networks

The system exists in two forms simultaneously:

**The neural network** — the source. The specifications, the tests, the
human-readable code. What is reasoned about, reviewed, and evolved.

**The neutral network** — the execution. The compiled agents running in
production, their source not exposed. What actually runs the deliberations.

The distinction is intentional. The thinking layer and the execution layer
are different expressions of the same intelligence.

## Why This Architecture Supports the Network

The network vision requires no architectural addition because the
architecture was built for it from the foundation:

- Agent-to-agent communication is native to the agent base, inherited by
  every agent in every organisation.
- The per-organisation mind is handled by the consciousness substrate,
  expressed in an interoperable format.
- The deliberation and resonance mechanisms operate identically whether
  within one Boardroom or across many.

The organism was designed, from the start, to scale beyond the single
organisation. The network is its natural expression.
